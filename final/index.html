Summary of Progress and Reflection

We built custom .dae files to test refraction and dispersion code we will write. We rendered the .dae file initially on blender and then figured out how to render the .dae file in our project 3 C++ code. Our main .dae file we will be testing with is a glass triangular prism inside a cornell box, which is shown below. Shown below are other custom .dae files we made and blender renders we made.

CBPrism.dae

CBPrism_angle.dae- prism on 25 degree angle

CBPrism.blend– rendered in blender using Cycles Render

CBDiamond.blend– rendered in blender using Cycles Render

We also worked on building a HSL to RGB color converter. We did this by developing color utility functions needed for dispersion such as HSLtoRGB(), RGBtoHSL(), and getWaveLength(). The code is shown below.

We also worked on getting functioning photon mapping software that we will use to create a dispersion effect. The source code of the photon mapping software can be found here.

Right now, we are on track to finish by the final deadline. Once the open source photon mapping software is up and running, we will be able to integrate the software and color conversion code into our project 3 code to test it out on our custom .dae files to show off our implemented dispersion.
